Me chamo Wilson, sou estudante de Web Design, em busca de criar experiências digitais que unem estética, funcionalidade e usabilidade. Ao longo da minha formação, venho desenvolvendo habilidades em HTML e CSS. Estou em constante evolução, aprendendo novas tecnologias. Meu objetivo é transformar ideias em interfaces intuitivas e visualmente impactantes, contribuindo com projetos que valorizam tanto a identidade visual quanto a experiência do usuário. O client-side code, ou código do lado do cliente, é a parte do desenvolvimento web que roda diretamente no navegador do usuário. Ele é responsável por tudo aquilo que o usuário vê e interage em uma página, como layouts, botões, animações e formulários. Utilizando linguagens como HTML, CSS e JavaScript, o client-side define tanto a estrutura visual quanto o comportamento dinâmico da interface, tornando a navegação mais fluida, intuitiva e atraente. Durante meus estudos em Web Design, quero entender e aplicar os princípios do client-side de forma eficiente, buscando sempre criar experiências agradáveis e funcionais. Essa base é essencial para construir acessíveis, além de garantir que o design pensado no papel seja fielmente traduzido na tela. O conceito de mobile first é uma abordagem de design e desenvolvimento que prioriza a criação de interfaces para dispositivos móveis antes de adaptá-las para telas maiores, como tablets e desktops. Essa metodologia parte do princípio de que a maioria dos usuários acessa a internet por meio de smartphones, exigindo interfaces leves, funcionais e otimizadas para telas pequenas. Ao começar o design pelo mobile, garantimos que o conteúdo mais importante esteja sempre acessível e que a navegação seja intuitiva, independentemente do tamanho da tela. Nos meus projetos como estudante de Web Design, quero aplicar a estratégia mobile first para garantir que cada layout seja responsivo desde o início. mantendo a coerência visual e a boa usabilidade. Essa prática tem sido fundamental para criar experiências mais inclusivas e eficientes, alinhadas com as necessidades reais dos usuários e as tendências atuais do design digital. UX (User Experience) e UI (User Interface) são dois pilares fundamentais no desenvolvimento de interfaces digitais. Enquanto o UX foca na experiência do usuário como um todo — desde a navegação até a satisfação final com o produto —, o UI está mais ligado à parte visual e interativa da interface, como cores, tipografia, botões e elementos gráficos. Um bom design precisa equilibrar esses dois aspectos para garantir que o usuário não só ache a interface bonita, mas também fácil de usar, intuitiva e funcional. Nos meus estudos e projetos, busco sempre aplicar os princípios de UX e UI de forma integrada. Começo entendendo as necessidades do usuário, organizando as informações de forma clara e pensando na jornada de navegação ideal. A partir disso, desenvolvo interfaces visuais que reforcem essa experiência, utilizando hierarquia visual, contraste e consistência. A combinação entre uma boa experiência e um visual bem planejado é essencial para criar soluções digitais que realmente façam sentido para quem as utiliza.
